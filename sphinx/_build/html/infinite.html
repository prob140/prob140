Infinite Distribution (prob140.InfiniteDistribution)¶

InfiniteDistribution Subclass of DiscreteDistribution to represent an Infinite Probability Distribution.
class prob140.InfiniteDistribution(labels=None, _deprecated=None, *, formatter=<datascience.formats.Formatter object>)[source]¶

Subclass of DiscreteDistribution to represent an Infinite Probability Distribution.

Construct an InfiniteDistribution by specifying both the range of the domain and a function for the probabilities

Examples

>>> geometric = InfiniteDistribution().domain(0,inf).probability_function(lambda k:0.4*(0.6)**k)
>>> geometric
